# Zacharia Kimani - Full-Stack Developer Portfolio

Modern, high-performance portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Features

- ⚡ Lightning fast with Vite
- 🎨 Modern, clean design
- 📱 Fully responsive
- ✨ Smooth animations with Framer Motion
- 🧭 React Router navigation
- 🎯 SEO optimized
- ♿ Accessible
- 🚀 Production ready

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`

## Deployment

The site is ready for deployment on Vercel or Netlify.

## Performance

Optimized for 90+ Lighthouse scores:
- Lazy loading images
- Code splitting
- Optimized animations
- Minimal dependencies
